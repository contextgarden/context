\chapter{Subscripts}

As we no longer typeset the consituents in math mode, we also no
longer need to worry about the alignment of super and subscripts.
We just use the same mechanism as in running text, using \type
{\low}, \type {\high} and \type {\lohi}. These are applied
automatically.

\stopcomponent
